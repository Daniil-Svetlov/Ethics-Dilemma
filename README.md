# 🤖 AI Ethics Interactive Case Study: The Database Disaster

An interactive, responsive web application designed to explore and analyze a critical professional dilemma: **human over-reliance on Artificial Intelligence and the ethical consequences of automated failures.**

Developed as a project for the English & Professional Ethics course.

## 📝 The Dilemma Overview
> *"I was given a task, but I used AI. I didn't notice how the AI deleted the entire database."*

This case study models a high-stakes corporate crisis where a junior developer blindly trusts an AI-generated script without proper code review or testing environments, resulting in catastrophic data loss. The project breaks down 6 potential choices (from unethical deflection to responsible engineering) and evaluates each through the lens of international and local ethical frameworks.

---

## ✨ Key Features
* **Interactive 3D Flashcards:** Users click or tap cards to flip them, revealing the exact ethical consequences hidden behind each choice.
* **Gamified Interface:** Choices are color-coded (Red for failures, Green for responsible solutions) only *after* interaction to maintain suspense.
* **Modern Dark Aesthetics:** Designed with a clean, developer-centric dark mode theme.
* **Fully Responsive:** Optimized for desktop hovers and seamless mobile touch gestures (iOS/Android compatible).
* **Pure & Lightweight:** Built using vanilla web technologies with zero external dependencies or heavy frameworks.

---

## 🛠️ Ethical Frameworks Applied
The consequences and feedbacks modeled in this application strictly align with:
1. **ACM Code of Ethics and Professional Conduct:**
   * *Section 1.3 (Be Honest and Trustworthy)* — analyzed in the "Hide the Mistake" scenario.
   * *Section 2.5 (High Standards of Professional Competence and Conduct)* — analyzed in the "Blame the AI" scenario.
2. **Russian Code of Ethics in the Field of Artificial Intelligence:**
   * *Article 4 (Risk Assessment and Responsible AI Implementation)* — applied to the preventive infrastructure solutions.
